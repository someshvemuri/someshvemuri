# Welcome to My Vault 🧠

This is my personal digital knowledge management system - my second brain. Here you'll find:

- **Notes** - My learning journey and knowledge base
- **Projects** - Project-specific documentation
- **Excalidraw** - Visual diagrams and sketches
- And more...

## Quick Navigation

- Explore the file tree on the left
- Search for notes using the search bar
- Click on tags to filter by topic
- View markdown content with full formatting

## Getting Started

1. Add your own notes to the vault folder
2. Use tags in your frontmatter for better organization
3. Link notes together with `[[wiki links]]`
4. Enjoy your knowledge graph! 📚
