# Portfolio Project

Building my personal portfolio website with Next.js and React.

## Objectives

1. Showcase my projects and experience
2. Create an Obsidian-like vault viewer
3. Implement interactive components
4. Optimize for performance

## Architecture

```
portfolio/
├── app/          - Next.js app directory
├── components/   - React components
├── lib/          - Utility functions
├── public/       - Static assets & vault
└── styles/       - CSS styles
```

## Stack

- **Frontend:** React 18, Next.js 14
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
